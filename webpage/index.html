Homelab

Single-node. Real infrastructure. Built to learn.

My homelab runs on a single Proxmox node. an Intel Core i7 11th Gen laptop with 32GB RAM, kept online 24/7 with the lid closed. It's where I test everything before it ever touches production.

Alongside it sits a self-built NAS (Intel N100, 8GB DDR5) handling local storage and backups via Proxmox Backup Server. All traffic goes through Cloudflare for DNS, CDN, and DDoS protection. this website included.

Services run in Docker inside LXC containers and VMs, managed and snapshotted directly from the Proxmox web UI. The entire setup is a single-host solution for now, with plans to expand into a multi-node cluster and add Kubernetes (K3s) as the lab grows.

This site is self-hosted here. No cloud provider, no managed hosting. just my own hardware in Switzerland.
