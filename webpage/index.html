Homelab

Two-node cluster. Real infrastructure. Built to learn.

My homelab is a two-node Proxmox VE 9 cluster built on a pair of Dell Precision 3561 laptops. Intel Core i7, 32 GB RAM each, kept online 24/7 with the lids closed. It's where I test everything before it ever touches production.

The two nodes are linked over a 40 GbE Thunderbolt 4 ring, with a Raspberry Pi acting as a QDevice for cluster quorum. not a full node, just the tiebreaker that keeps HA voting honest with an even node count.

Sitting next to the cluster, an OpenMediaVault NAS serves as the backup target. it lives outside the Proxmox cluster on purpose, so a cluster-wide failure can't take the backups with it. All public traffic exits through Cloudflare for DNS, CDN and DDoS protection.

A custom power-resilience system turns the laptop batteries into a built-in UPS, so the cluster rides through outages without a separate battery box. Services run in Docker inside LXC containers and VMs, snapshotted and migrated from the Proxmox web UI.

This site is self-hosted here. No cloud provider, no managed hosting. just my own hardware in Switzerland.
